Project Title:
Image Object Detection System using AWS and YOLOv5

Description:
This project uses Amazon S3, Amazon EC2, AWS Lambda, API Gateway, and YOLOv5 to perform image object detection. Images are uploaded to S3, processed using YOLOv5 on EC2, and the output is uploaded back to S3.

AWS Services Used:
- Amazon S3
- Amazon EC2
- AWS Lambda
- Amazon API Gateway
